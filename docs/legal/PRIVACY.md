# プライバシーポリシー（Privacy Policy）

最終更新日: 2026年6月26日

本プライバシーポリシーは、本アプリ「Study English」（以下「本アプリ」）における利用者の情報の取り扱いについて、本アプリの開発者（以下「当方」）が定めるものです。

## 1. 基本方針

本アプリは、利用者の入力した英文の添削を行う PWA アプリです。データは原則として利用者の端末内に保存されますが、ログイン時には一部のデータがクラウドに同期され、また添削のために一部のデータが Google の AI サービスに送信されます。本ポリシーは、これらの取り扱いを説明するものです。

## 2. 端末内に保存されるデータ

本アプリのデータ（添削履歴・ミス記録・設定・APIキー等）は、利用者の端末のブラウザの **localStorage** に保存されます。当方は、これらのデータを直接収集・閲覧することはありません。

## 3. クラウド同期（Firebase）

1. 利用者が**ログインした場合**、添削セッションのデータが **Google が提供する Firebase Authentication および Cloud Firestore** に保存・同期されます。
2. クラウドに保存されるデータは、Firestore のセキュリティルールにより、**ログインした本人（自身の UID）のみ**が読み書きできるよう制限されています。
3. これらのデータは Google のインフラ上に保存され、その取り扱いには **Google のプライバシーポリシー**が適用されます。当方は Google によるデータの取り扱いについて責任を負いません。
4. ログインしない場合、クラウド同期は行われません。

## 4. AI 添削のためのデータ送信（Gemini API）

1. 利用者が添削機能を利用する場合、本アプリは利用者が入力した英文等を、利用者自身が設定した API キーを用いて **Google Gemini API** に送信します。
2. 送信されたデータの取り扱いは、Google のプライバシーポリシーおよび利用規約に従います。当方はこれらのデータの取り扱いについて責任を負いません。

## 5. API キーの取り扱い

利用者が設定した Gemini API キーは、利用者の端末内（localStorage）に保存され、当方のサーバーに送信されることはありません。API キーの管理は利用者の責任で行ってください。

## 6. Cookie・アクセス解析

本アプリは、利用者を追跡する目的での Cookie やアクセス解析ツールを使用しません。なお、Firebase 等の第三者サービスが、そのサービス提供に必要な範囲で識別子等を利用する場合があります。

## 7. お問い合わせ

本ポリシーに関するお問い合わせは、本アプリのリポジトリの Issues 等を通じてお願いします。

## 8. 本ポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。変更後の本ポリシーは、本アプリまたはリポジトリ上に掲示した時点で効力を生じます。
