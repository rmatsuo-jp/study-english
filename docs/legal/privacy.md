# プライバシーポリシー（Privacy Policy）

最終更新日: 2026年7月8日

本プライバシーポリシーは、本アプリ「英文ラボ（Eibun-Lab）」（以下「本アプリ」）における利用者の情報の取り扱いについて、本アプリの開発者（以下「当方」）が定めるものです。

## 1. 基本方針

本アプリは、利用者の入力した英文の添削を行う PWA アプリです。データは原則として利用者の端末内に保存されますが、当方が許可した利用者がログインした場合には一部のデータがクラウドに同期され、また添削のために一部のデータが Google の AI サービスに送信されます。本ポリシーは、これらの取り扱いを説明するものです。

## 2. 端末内に保存されるデータ

1. 本アプリのデータ（添削履歴・ミス記録・設定等）は、利用者の端末のブラウザの **localStorage** に保存されます。当方は、これらのデータを直接収集・閲覧することはありません。
2. Gemini API キーは平文では保存せず、**Web Crypto API（AES-GCM）で暗号化**したうえで端末内に保存します（暗号鍵は取り出し不可能な形式でブラウザの IndexedDB に保持します）。
3. 端末内のデータは、ブラウザの設定・仕様（サイトデータの削除、ストレージ容量の逼迫、一部ブラウザにおける長期間未使用サイトのデータ自動削除等）により、**予告なく削除される場合があります**。重要なデータは本アプリのエクスポート機能で定期的にバックアップすることを推奨します。
4. 端末内のデータは、**同じブラウザプロファイルを利用する他の人からも閲覧・操作が可能**です。共用端末・公共の端末での利用にあたっては、利用後にサイトデータを削除する等、利用者自身で適切に管理してください。

## 3. クラウド同期（Firebase）

1. クラウド同期機能は、第三者による無断利用を防止するため、**当方が個別に許可した利用者のみ**が利用できます。許可されていない利用者は、ログインしてもクラウドへのデータの保存・同期は行われません（データは端末内にのみ保存されます）。
2. 許可された利用者が**ログインした場合**、添削セッションおよびドリル進捗のデータが **Google が提供する Firebase Authentication および Cloud Firestore** に保存・同期されます。
3. クラウドに保存されるデータは、Firestore のセキュリティルールにより、**許可された本人（自身の UID）のみ**が読み書きできるよう制限されています。
4. これらのデータは Google のインフラ上に保存され、その取り扱いには **Google のプライバシーポリシー**が適用されます。当方は Google によるデータの取り扱いについて責任を負いません。
5. ログインしない場合、クラウド同期は行われません。
6. ログイン（Google アカウントによるサインイン）を行うと、許可の有無にかかわらず、Firebase Authentication に **Google アカウントのメールアドレス・表示名・プロフィール画像・内部識別子（UID）** の記録が作成され、当方が管理画面上で閲覧できる状態になります。当方はこれらの情報を、クラウド同期の利用可否の判定および不正利用の防止の目的にのみ利用し、それ以外の目的には利用しません。許可されていない利用者のアカウント記録は、当方が認識した時点で削除します。削除を希望される場合は「お問い合わせ」記載の方法でご連絡ください。

## 4. AI 添削のためのデータ送信（Gemini API）

1. 利用者が添削機能を利用する場合、本アプリは利用者が入力した英文等を、利用者自身が設定した API キーを用いて **Google Gemini API** に送信します。
2. 送信されたデータの取り扱いは、Google のプライバシーポリシーおよび利用規約に従います。当方はこれらのデータの取り扱いについて責任を負いません。

## 5. API キーの取り扱い

1. 利用者が設定した Gemini API キーは、**Web Crypto API（AES-GCM）で暗号化したうえで利用者の端末内にのみ保存**され、当方のサーバーに送信されることはありません。
2. 暗号化は端末内での保護を強化するものであり、あらゆる攻撃に対する完全な保護を保証するものではありません。API キーの管理（不要になった際の削除・無効化、利用上限の設定等を含みます）は利用者の責任で行ってください。

## 6. データの削除

1. 端末内のデータは、本アプリの設定画面またはブラウザのサイトデータ削除機能により、利用者自身でいつでも削除できます。
2. クラウドに同期されたデータについて、本アプリ内での削除操作は多端末間で削除を同期させるための**削除記録（論理削除）**として扱われるため、削除操作後もクラウド上に削除記録が残る場合があります。クラウド上のデータの完全な削除（アカウント記録の削除を含みます）を希望される場合は、「お問い合わせ」記載の方法でご連絡ください。合理的な期間内に削除します。

## 7. Cookie・アクセス解析

本アプリは、利用者を追跡する目的での Cookie やアクセス解析ツールを使用しません。なお、Firebase 等の第三者サービスが、そのサービス提供に必要な範囲で識別子等を利用する場合があります。

## 8. 未成年者の利用

未成年者が本アプリを利用する場合は、親権者等の法定代理人の同意を得たうえで利用してください。

## 9. お問い合わせ

本ポリシーに関するお問い合わせ、およびデータ・アカウント記録の削除のご依頼は、本アプリのリポジトリの Issues 等を通じてお願いします。**Issues は誰でも閲覧できる公開の場です。** メールアドレス・API キーその他の個人情報や秘密情報は Issues に記載しないでください（削除依頼の本人確認が必要な場合は、当方から非公開の連絡手段を個別に案内します）。

## 10. 本ポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。変更後の本ポリシーは、本アプリまたはリポジトリ上に掲示した時点で効力を生じます。
